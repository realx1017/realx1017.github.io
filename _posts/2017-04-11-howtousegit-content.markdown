**git 사용 시 history 조회 및 변경 내용을 보고 싶을 때**

#. git add filename

#. git commit -m "comment"

#. git push origin branch name

#. git log -p -1 <- 가장 최근의 변경 내용 확인

#. git show commitid <- commit 로 파일의 변경 내용 확인
